How the Clock Is Set

末日钟是一种诊断，不是预测。这里的 midnight 表示资本耐心比技术闭环先耗尽：融资撤场、估值重定价、PR 叙事反噬。

铁轨 = research / lab progress：VLA 控制、world model、真实数据飞轮、synthetic data、on-device 推理、跨 embodiment 泛化、会议和 arXiv 的问题密度。

火车 = capital / PR pressure：融资速度、估值斜率、mega-round、发布会承诺、household / factory deployment narrative，以及“今年就能落地”的语气强度。

当前读数 11:42 PM = rail 64, train 82。差距 18 点，所以离 midnight 18 分钟。若铁轨超过火车，钟会往回拨；若资本火车继续加速而技术闭环停滞，钟会靠近 midnight。

What would move the clock back?

可靠的跨场景部署、可复现的 robot data engine、公开 benchmark 上的长期任务成功率、真实客户的 uptime/cost 数据。

What moves it forward?

融资和估值继续倍增，但实际交付仍停留在 demo、beta waitlist 或少量 pilot；发布会承诺越具体，耐心消耗越快。

What is derailment?

不是技术失败，而是资本撤场：估值重定价、下一轮断裂、客户不续约、媒体从“具身智能新工业革命”转向“机器人泡沫”。
